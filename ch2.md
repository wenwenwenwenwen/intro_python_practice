## 資料: 型態、值、變數與名稱
2.1 將整數值`99`指派給變數prince，並印出它
    ```python
    >>> prince = 99
    >>> prince # print(prince)
    99
    >>>
    ```

2.2 值`5`的型態是?
    ```python
    >>> type(5)
    <class 'int'>
    >>>
    ```

2.3 值`2.0`的型態是?
    ```python
    >>> type(2.0)
    <class 'float'>
    >>>
    ```

2.4 運算式`5 + 2.0`的型態是?
    ```python
    >>> type(5+2.0)
    <class 'float'>
    >>>
    ```
